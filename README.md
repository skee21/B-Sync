# B Sync

B Sync is a visually immersive Flutter application designed with advanced theming and interactive visual effects. Built with performance and aesthetics in mind, it offers a highly customizable user experience ranging from Glassmorphism to Cyberpunk aesthetics.

## Features

### Advanced Theming
Customize the look and feel of the application with distinct style presets:
- **Normal:** Clean and standard interface.
- **Glass:** Modern glassmorphism with rising bubble effects.
- **Nightcore:** Energetic aesthetic with hearts, stars, and particle effects.
- **Cyberpunk:** Futuristic 2077-inspired look with glitch effects.
- **High Contrast:** Accessibility-focused with scanline overlays.

### Visual Effects
- **Interactive Particles:** Floating particles, rising bubbles, and dynamic background elements.
- **Live Preview:** Real-time preview of theme changes and color profiles within the settings.
- **Animations:** Smooth transitions and entrance animations powered by `flutter_animate`.

### Customization
- **Color Profiles:** Override theme colors with specific presets or use dynamic system colors.
- **Theme Modes:** Full support for Light, Dark, and System modes.

## Tech Stack

- **Framework:** Flutter
- **State Management:** Riverpod
- **Animations:** flutter_animate

## Getting Started

### Prerequisites
- Flutter SDK (Latest Stable)
- Dart SDK
- Android SDK 34.0.0
